# Foro-Hub
Pasos para utilizar este proyecto:
Descargar el Archivo .Zip.
Descomprimir.
Utilizar el IDE de preferencia.
Asegurarce que las depencencias cargen correctamente.
Antes de iniciar la app configurar las varibles de entorono con los datos de su servidor, pueden utilizar el que ustedes elijan.
Tener un progrma para testear como Postman o Insomnia, puede ser cualquiera.
Hacer las solicitudes correspondientes.
Iniciar la app y correr las solicitudes.
